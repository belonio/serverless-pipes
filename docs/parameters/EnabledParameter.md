# Usage
```yaml

# serverless.yml

pipes:
  pipeName:
    enabled: true
```


### enabled


It defines the `DesiredState` of EventBridge Pipes. If set to true sets the `DesiredState` property to `RUNNING` else sets to `STOPPED`.
